## 一、原型概述

### 1.1 原型目标
基于提示词要求，构建了包含完整页面和交互的高保真HTML原型，支持：
- 登录认证
- 案例导入与自动拆解
- 路径概率图谱可视化（正向迁移/逆向跳转区分）
- 实时话术引导
- 数据报表展示

### 1.2 技术栈
- HTML5 + CSS3 + JavaScript (ES6+)
- D3.js v7 - 图谱可视化
- SheetJS - Excel/CSV解析
- 纯前端实现，无需后端即可演示

## 三、核心功能实现

### 3.1 正向迁移 vs 逆向跳转区分

| 路径类型 | 视觉标识 | 颜色 | 线宽规则 |
|----------|----------|------|----------|
| 正向迁移 (A/B/C) | 绿色实线 | #4CAF50 | ≥70%=4px, 50-69%=2px, 30-49%=1.5px |
| 逆向跳转 (D) | 橙色虚线 | #FF9800 | ≥30%=3px, 10-29%=2px, <10%=1px |

### 3.2 Stage颜色体系

| Stage | 颜色 | 色值 |
|-------|------|------|
| S1 开场破冰 | 蓝色 | #1E88E5 |
| S2 需求深挖 | 青色 | #00ACC1 |
| S3 价值传递 | 绿色 | #4CAF50 |
| S4 顾虑化解 | 黄色 | #FFC107 |
| S5 邀约收口 | 紫色 | #9C27B0 |

### 3.3 数据同步效果

- 文件上传后自动解析
- 生成Stage-Topic标注
- 右上角弹出同步成功通知
- 显示新增正向/逆向路径数量
